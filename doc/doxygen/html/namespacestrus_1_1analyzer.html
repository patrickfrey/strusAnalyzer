enum strus::analyzer::PositionBind

Determines how document ordinal positions are assigned to terms.

Enumerator
BindContent 

An element in the document that gets an own ordinal position assigned.

BindSuccessor 

An element in the document that gets the ordinal position of the element at the same position or the succeding content element assigned.

BindPredecessor 

An element in the document that gets the ordinal position of the element at the same position or the preceding content element assigned.
