pure virtual

Define a pattern for detecting a basic lexem of this pattern matching lexer.

Parameters
[in]ididentifier given to the lexem, 0 if the lexem is not part of the output (only used for assigning ordinal positions).
[in]expressionexpression string defining the lexem
[in]resultIndexindex of subexpression that defines the result lexem, 0 for the whole match
[in]levelweight of this lexical pattern. A lexical pattern match causes the suppressing of all lexems of lower level that are completely covered by one lexem of this pattern
[in]posbinddefines how the ordinal position is assigned to the result lexem
Remarks
For performance it may be significantly better to define rules without resultIndex selection. But sometimes you need pattern sub expression selections for defining the result.
